

# Awana Labs - Open Source Projects Showcase

A sleek, single-page application to display Awana Labs' open-source projects with elegant animations and a professional design.

---

## Brand Identity

- **Primary Color:** Deep plum (#4C1130)
- **Accent Color:** Warm cream (#F7F3EB)
- **Typography:** Clean sans-serif (Inter font family)

---

## Page Sections

### 1. Hero Section
- Full-viewport opening with animated topographic contour line background
- Organization name, tagline, and brief mission description
- Subtle parallax scroll effect on the contour lines
- Animated scroll indicator to guide users down

### 2. Projects Gallery
- **Search bar** to filter by title, description, or tags
- **Pill button filters** for project status (All, Active, Paused, Archived)
- Responsive card grid (1-3 columns depending on screen size)
- Cards animate in with staggered reveal as they enter viewport

### 3. Project Cards
Each card displays:
- Project logo (or styled text placeholder if no logo)
- Title and truncated description
- Tag badges (colored pills)
- Status indicator badge
- Hover effect: subtle lift and shadow enhancement

### 4. Project Detail Modal
Clicking a card opens a full-detail modal featuring:
- Large logo/header area
- Complete description
- Image carousel with navigation arrows
- All tags displayed
- Status and usage badges
- Quick-access buttons for Homepage, Repository, and Documentation links
- Smooth scale-and-fade animation on open/close
- Keyboard accessible (Escape to close, focus trap)

### 5. Footer
With :heart: by Awana Digital

---

## Animation & Accessibility

- All animations powered by Framer Motion
- Respects `prefers-reduced-motion` for users who prefer minimal animation
- Full keyboard navigation support
- ARIA labels and focus management for modal
